Expand description

Extension trait to convert toml_edit::Value into Arcella’s canonical Value.

Only TOML scalar types and arrays of scalars are supported. The following TOML types are not supported and will cause an error:

  • Datetime
  • Inline tables

Arrays are supported recursively, but must contain only supported scalar types.

§Errors

Returns ArcellaUtilsError::TOML if the value contains an unsupported type (e.g., datetime, inline table, or nested unsupported structure).
