pub async fn find_base_dir() -> Result<PathBuf>
Expand description

Determines the base directory for Arcella based on the executable location or environment.

The function follows this priority order:

  1. If the executable is located in a bin subdirectory, the parent of bin is used.
  2. If the current directory (where the executable is run from) contains a config subdirectory, the current directory is used.
  3. Otherwise, the user’s home directory joined with .arcella is used.

§Returns

A Result containing the determined PathBuf or an error if the home directory cannot be determined.
